# Endless-Runner

## About

Endless Runner is a game made in Unity. The player controls a block that has to dodge other falling blocks. The goal of the game is to survive as long as possible.

## Features

The game has:
- 3 enemy types
- highscore tracking
- changing background based on score
- sound
- menu screen
- options menu
- pause menu

## How to play

### `Left arrow key - move left`
### `Right arrow key - move right`
### `Escape key - pause game`
